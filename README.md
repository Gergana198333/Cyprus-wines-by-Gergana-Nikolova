# Cyprus Wines by Gergana Nikolova

A beautiful, elegant website showcasing the rich wine heritage and exceptional vineyards of Cyprus.

## Project Overview

This website is dedicated to exploring Cyprus's remarkable wine history, from ancient traditions to modern innovations. Created by Gergana Nikolova as part of her research project "From Vineyard to Glass," this site educates wine enthusiasts about:

- **Ancient Wine Heritage**: Cyprus's thousands of years of winemaking tradition
- **Wine Regions**: Troodos Mountains, Commandaria, Paphos, and Larnaca regions
- **Indigenous Varieties**: Mavro, Xynisteri, Maratheftiko, Yiannoudi, Promara, and Ofthalmo
- **Featured Wines**: Legendary Commandaria to contemporary expressions
- **Wine Production**: The complete journey from vineyard to glass

## Features

### Design
- **Elegant, Modern Aesthetic**: Inspired by the Jancis Robinson wine journalism website
- **Wine & Earth Color Palette**: Rich wine colors (#722f37), gold accents (#d4af37), and cream backgrounds
- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Premium Typography**: Serif fonts for headings, sans-serif for body text
- **Smooth Animations**: Subtle, elegant transitions and hover effects

### Content Sections
1. **Navigation Bar**: Sticky navigation with smooth scrolling
2. **Hero Section**: Striking introduction with floating wine glass animation
3. **Introduction**: Overview of Cyprus wine culture
4. **Wine History**: Four historical periods of Cypriot winemaking
5. **Wine Regions**: Detailed information about all major wine regions
6. **Indigenous Varieties**: In-depth profiles of unique Cypriot grapes
7. **Featured Wines**: Showcase of exceptional Cypriot wines
8. **Vineyard to Glass**: The production process explained
9. **Why Cyprus Wines**: Six compelling reasons to explore Cyprus wines
10. **About Gergana**: Information about the researcher and links to her work
11. **Footer**: Navigation and additional resources

## How to Use

1. Open `index.html` in a web browser
2. Navigate through sections using the sticky navigation bar
3. Click on any navigation link to jump to that section
4. Explore information about Cyprus's wine regions, grape varieties, and wines
5. Visit the footer for additional resources and links

## Responsive Breakpoints

- **Desktop**: Full multi-column layouts
